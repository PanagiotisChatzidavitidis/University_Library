Συμαντικά σχόλια:
1) Για το σετάρισμα του docker: Στο CMD κάνουμε cd στον φάκελο της εργασίας και γράφουμε την ενολή 'docker-compose up'.
2) Συνδέεστε στην αρχική σελίδα : http://localhost:5000/
3) Τα credentials του Διαχειριστή είναι: [email:admin@gmail.com και password:admin]
4) Tο λειτουργικο συστημα του υπολογιστη μου ειναι Windows 11
